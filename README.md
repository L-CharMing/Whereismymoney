# Where Is My Money

一个面向 Android 15+ 的记账 App 原型，重点覆盖：

- 在支付完成后自动抓取账单候选数据（无障碍优先，ADB 可选）
- 按用户规则决定哪些账单自动记录、哪些忽略、哪些需要确认
- 预留大模型 / 外部 API 分类接口
- 按月汇总支出结构，展示“钱都花去哪了”

## 当前原型包含

1. **Jetpack Compose 三页原型界面**
   - 总览页：按月统计支出分类和简要建议
   - 自动记账页：展示支付采集链路与规则命中结果
   - 规则 / AI 页：展示过滤规则和模型接口配置入口
2. **自动记账扩展点**
   - `PaymentAccessibilityService`：用于识别支付成功页或详情页
   - `CaptureSettings`：可打开/关闭无障碍、ADB、通知镜像等链路
3. **可替换的分类器接口**
   - `BillClassifier`：本地规则分类接口
   - `RemoteClassifierGateway`：远程 LLM/API 网关接口
4. **月度总结能力**
   - `monthlySummary()`：按分类聚合金额
   - `monthlyInsight()`：输出本月支出重点和提醒建议

## 设计取舍

- 当前仓库为空，因此先搭建一个**可运行方向明确的 Android 原型结构**。
- 真实“自动记账”涉及支付应用适配、权限引导、隐私合规、异常兜底与本地数据库；这些会在下一轮补上。
- Android 不存在给普通应用直接“获取 ADB 权限”的稳定官方方式，因此这里的设计是：
  - 手机端支持无障碍自动抓取；
  - 如用户愿意，可通过桌面端 ADB/调试桥接辅助开发和调试；
  - 是否启用这类高级能力，必须经用户明确同意。
