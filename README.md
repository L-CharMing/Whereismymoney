# Where Is My Money

一个本地优先的 Android 15+ 记账 App 原型：

- 默认把账单、规则、设置都保存在本地文件中
- 支持手动记账、规则过滤、月度汇总
- 预留无障碍 / ADB 辅助自动记账入口
- 仅在你接入大模型分类时才需要联网

## 当前能力

1. **本地账本持久化**
   - 通过 `JsonLedgerStorage` 把账单快照保存到 App 私有目录
   - App 重启后仍然保留账单、规则和 AI 配置
2. **可交互原型界面**
   - 总览页：查看本月总支出、分类占比、最近账单，并支持手动新增账单
   - 自动记账页：切换无障碍 / ADB / 通知镜像，并可导入模拟支付账单
   - 规则 / AI 页：保存 AI 接口配置、新增账单匹配规则
3. **本地优先数据流**
   - 所有汇总、规则命中、账单保存都在本地完成
   - 大模型接口只作为后续分类增强能力

## 设计原则

- **本地优先**：除大模型调用外，不依赖联网。
- **用户可控**：用户决定哪些账单记、哪些忽略、哪些需要确认。
- **渐进增强**：先把手动记账 + 规则 + 总结做扎实，再逐步接入自动抓取。

## 后续建议

- 接 Room 替代 JSON 文件，支持更稳定的本地查询与迁移
- 增加账单编辑、删除、搜索、按月切换
