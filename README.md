# Where Is My Money

一个本地优先的 Android 15+ 记账 App 原型：

- 默认把账单、规则、设置都保存在本地文件中
- 支持手动记账、编辑账单、规则过滤、月度汇总
- 预留无障碍 / 无线 ADB 辅助自动记账入口
- 仅在你接入大模型分类时才需要联网

## 当前能力

1. **本地账本持久化**
   - 通过 `JsonLedgerStorage` 把账单快照保存到 App 私有目录
   - App 重启后仍然保留账单、规则和 AI 配置
2. **可交互原型界面**
   - 总览页：查看本月总支出、分类占比、最近账单，并支持手动新增与编辑账单
   - 自动记账页：切换无障碍 / 无线 ADB / 通知镜像，并可导入模拟支付账单
   - 规则 / AI 页：保存 AI 接口配置、新增账单匹配规则
3. **本地优先数据流**
   - 所有汇总、规则命中、账单保存都在本地完成
   - 大模型接口只作为后续分类增强能力

## 支付软件接入思路

- 微信 / 支付宝更现实的官方开放能力主要面向**商户接入支付**，不适合作为个人记账 App 直接读取用户个人账单的通用接口。
- 因此当前产品方向调整为：
  - **首选无障碍识别支付成功页**；
  - **无线 ADB 作为开发调试或高级用户辅助能力**；
  - 如后续发现特定官方账单导出 / 授权能力，再以“用户明确授权 + 本地优先缓存”的方式接入。

详见 `docs/auto-capture-plan.md`。

## 设计原则

- **本地优先**：除大模型调用外，不依赖联网。
- **用户可控**：用户决定哪些账单记、哪些忽略、哪些需要确认。
- **渐进增强**：先把手动记账 + 规则 + 总结做扎实，再逐步接入自动抓取。

## 后续建议
