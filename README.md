# Where Is My Money

一个本地优先的 Android 15+ 记账 App 原型：

- 首页只展示当月明细和统计
- 手动记账通过按钮弹窗录入金额、用途、商户、分类
- 自动记账、AI记账、导入导出统一放到“设置”页
- 自动记账以无障碍为主，ADB 作为高级功能
- 所有账单数据默认保存在本地

## 页面结构

1. **首页**
   - 按月切换
   - 当月总支出与分类统计
   - 当月账单明细列表
   - 手动记账按钮（弹窗录入）
2. **设置**
   - 自动记账设置（无障碍开关、跳转系统无障碍设置、白名单、去重窗口）
   - AI 记账设置（Endpoint / Token）
   - 导入 / 导出（导出当前月份 CSV）

## 无障碍与 ADB 说明

- 无障碍服务入口名称为 `Where Is My Money 自动记账服务`，应在系统无障碍列表中更容易识别。
- 若找不到服务，请确认：
  - App 已成功安装且版本为最新；
  - 手机系统版本满足应用最低版本要求（当前配置 `minSdk=35`）。
- ADB 仅作为高级调试能力，不是默认主链路。
