# Where Is My Money

一个本地优先的 Android 15+ 记账 App 原型：

- 默认把账单、规则、设置都保存在本地文件中
- 支持手动记账、编辑账单、删除账单、搜索账单、按月切换、月度汇总
- 以无障碍作为自动记账主链路，ADB 仅作为高级功能开放
- 对自动抓取增加白名单和本地去重
- 仅在你接入大模型分类时才需要联网

## 当前能力

1. **本地账本持久化**
   - 通过 `JsonLedgerStorage` 把账单快照保存到 App 私有目录
   - App 重启后仍然保留账单、规则和 AI 配置
2. **账本可用性增强**
   - 总览页支持新增、编辑、删除账单
   - 支持按标题 / 商户 / 分类搜索
   - 支持按月切换查看汇总和账单
3. **自动记账主链路**
   - 默认以无障碍为主
   - ADB 只作为开发调试 / 高级功能开放
   - 对自动抓取增加包名白名单和时间窗去重
4. **本地优先数据流**
   - 所有汇总、规则命中、账单保存都在本地完成
   - 大模型接口只作为后续分类增强能力

## 设计原则

- **本地优先**：除大模型调用外，不依赖联网。
- **用户可控**：用户决定哪些账单记、哪些忽略、哪些需要确认。
- **无障碍优先**：把自动记账建立在更稳定的系统能力之上。
- **渐进增强**：先把手动记账 + 自动抓取 + 去重做扎实，再逐步增加更复杂能力。
